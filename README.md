# 天气API插件

一个基于 AstrBot 的天气查询插件，支持文字/图片天气查询，每日定时推送，多用户任务管理。

---

## 功能介绍

- 📝 调用柠柚API获取**文字**天气
- 🖼️ 调用柠柚API获取**图片**天气
- 📅 支持查询**当天**或**多天预报**
- ⏰ 每日**定时推送**天气（所有人可添加自己的任务）
- 🔒 管理员可**删除任何人**的定时任务
- 💾 定时任务**持久化**保存，Bot 重启后自动恢复

---

## 安装方法

将插件文件夹放到 AstrBot 的 `data/plugins/` 目录下，重启 AstrBot 即可自动加载。

```
data/
└── plugins/
    └── astrbot_plugin_api_caller/
        ├── main.py
        ├── metadata.yaml
        ├── _conf_schema.json
        └── README.md
```

---

## 配置

在 AstrBot WebUI → 插件 → 找到本插件 → 点击**插件配置**，填写以下配置项：

| 配置项 | 说明 | 默认值 |
|---|---|---|
| `API 地址` | 天气 API 的请求地址 | `https://api.nycnm.cn/API/weather.php` |
| `API Key` | 接口鉴权密钥 | 留空 |
| `默认返回格式` | `image` 返回图片，`text` 返回文字 | `image` |

> 💡 本插件默认使用 **柠柚API** 的天气接口。
>
> 如果服务端要求密钥，请前往 [https://api.nycnm.cn](https://api.nycnm.cn) 注册账号后获取 API 密钥，填入插件配置的 `API Key` 中即可。

## 指令说明

### 📢 所有人可用

| 指令 | 说明 | 示例 |
|---|---|---|
| `/天气 <城市> [天数]` | 查询天气（走默认格式配置） | `/天气 北京` |
| `/天气text <城市> [天数]` | 强制返回文字天气 | `/天气text 北京 5` |
| `/天气img <城市> [天数]` | 强制返回图片天气 | `/天气img 上海` |
| `/天气帮助` | 查看插件使用说明 | `/天气帮助` |
| `/定时 add <HH:MM> <text\|image> <城市> [天数]` | 添加每日定时推送 | `/定时 add 08:00 image 北京` |
| `/定时 del <ID> [ID2...]` | 删除自己的定时任务（支持批量） | `/定时 del 1 2 3` |
| `/定时 list` | 查看自己的定时任务 | `/定时 list` |

### 🔒 管理员额外权限

| 指令 | 说明 |
|---|---|
| `/定时 list` | 查看**所有人**的定时任务（含添加者信息） |
| `/定时 del <ID>` | 删除**任何人**的定时任务 |

## 使用示例

**查询今日天气：**
```
用户：/天气 北京
Bot：⏳ 图片获取中...
Bot：[天气图片]
```

**查询5天文字预报：**
```
用户：/天气text 北京 5
Bot：⏳ 查询中...
Bot：📍 北京 5天天气预报
     ...
```

**添加每日定时推送：**
```
用户：/定时 add 08:00 image 北京
Bot：✅ 定时任务已添加
     ID：1
     时间：每天 08:00
     类型：image
     城市：北京
```

**添加5天预报定时推送：**
```
用户：/定时 add 08:00 image 北京 5
Bot：✅ 定时任务已添加
     ID：2
     时间：每天 08:00
     类型：image
     城市：北京
     天数：5天
```

**批量删除任务：**
```
用户：/定时 del 1 2 3
Bot：✅ 已删除任务：1, 2, 3

用户：/定时 del 1,2,3
Bot：✅ 已删除任务：1, 2, 3
```

**查看任务列表（普通用户只看自己的）：**
```
用户：/定时 list
Bot：📋 定时任务列表：
     ID：1  时间：每天 08:00  类型：image  城市：北京
```

**管理员查看所有任务：**
```
管理员：/定时 list
Bot：📋 定时任务列表：
     ID：1  时间：每天 08:00  类型：image  城市：北京  添加者：123456
     ID：2  时间：每天 20:00  类型：text   城市：上海  添加者：654321
```

---

## 注意事项

- 定时任务保存在 `data/plugins_data/astrbot_plugin_api_caller/tasks.json`，Bot 重启后**自动恢复**
- 定时消息会发送到**执行添加指令时的会话**（群聊或私聊）
- 时间以**服务器本地时间**为准，请确认服务器时区正确
- 图片获取失败时会**自动降级为文字**发送
- 普通用户只能删除**自己添加**的任务，管理员可删除所有任务
